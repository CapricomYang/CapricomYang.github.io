Linux命令集合

常用命令

cd  / ./  ../ 			进入 根目录 当前目录 上一层目录   
cat 					查看文件
vi vim                  编辑文件
scp                     从其他服务器拷贝文件
cp                      拷贝文件
mkdir                   创建文件夹

日志相关

tail -f xxx.log 		查看日志 实时刷新显示尾部
grep '2021-11-22 11:1[3-4]' xxx.log  查看 log日志文件 2021年11月22日 11点 13分钟开始到14分钟结束的所有日志 前缀日期格式取决于日志文件格式
    
cat 文件名 |head -n 截止第几行 | tail -n +从第几行开始
如：cat xxx.log |head -n 20000 | tail -n +19990 查看第19990行到20000行之间的日志

服务器相关

nohup                  需要执行的命令 &  执行结果输出于nohup.out 避免会话结束终止进程
  
sudo lsof -i:8081 
sudo kill -9 “PID”     查询端口号为8081的进程 并根据查询出的PID 结束进程

其他命令

du -h ./ 				查看当前文件夹 及内部文件及文件夹大小
df -h 					查看磁盘大小使用情况
